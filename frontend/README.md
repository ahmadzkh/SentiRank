# SentiRank Frontend

This folder contains the FE-06 NextJS foundation for the SentiRank thesis frontend.

## Stack

- NextJS App Router
- TypeScript
- Tailwind CSS
- ESLint
- shadcn/ui configuration
- lucide-react
- recharts
- clsx
- tailwind-merge
- class-variance-authority

## Setup

```bash
npm install
npm run dev
```

Development server:

```txt
http://localhost:3000
```

## Checks

```bash
npm run lint
npm run build
```

## Scope

FE-06 only sets up the frontend foundation. Dashboard pages, mock data, full layout implementation, and API integration are intentionally deferred to later phases.

Use `frontend/DESIGN.md` as the canonical design specification before adding components or pages.
